# 01 - Overview

This document provides a high-level summary of the Investment Platform.

## System Purpose
The platform is a multi-level marketing (MLM) investment application that manages financial cycles, capital investments, downline agent hierarchies, and multi-tier commissions. It acts as an intermediary system between three primary user roles, driven entirely by Admin approvals.

## Core Features
- **User Roles:** Admin, Agent, Investor.
- **Investments:** Fixed monthly ROI returns based on initial deposits and top-ups.
- **Commissions:** Automatically distributed up to 4 levels of agents when their downline invests.
- **Transaction Processing:** Manual approval workflow for all Deposits and Withdrawals.
- **Agent Tree:** Network hierarchy mapping to 4 tiers.
